# 프로젝트 개요

## 목적

KHU:DArchive는 사용자의 커리어 기록을 경험 단위로 구조화해 저장하고, 향후 자기소개서 작성, 직무별 경험 추천, 면접 준비에 재사용할 수 있는 개인 경험 저장소를 만드는 프로젝트입니다.

## 핵심 기능

- 커리어 기록 입력
- 원문 기록 보존
- 경험 단위 정리
- 경험별 출처 추적
- 부족한 정보를 채우기 위한 보완 질문
- 경험 기반 검색
- OpenAPI 기반 API 명세 관리

## 화면 구성

프론트엔드 화면은 다음 흐름을 기준으로 설계합니다.

- 기록 입력 화면: 자기소개서, 이력서, 포트폴리오 메모 등을 입력
- 처리 결과 화면: 추출된 경험 목록과 완성도 확인
- 경험 상세 화면: STAR 구조, 기술 스택, 출처, 보완 질문 확인
- 보완 질문 화면: 부족한 역할, 성과, 수치, 학습 내용을 답변
- 검색 화면: 직무나 역량 키워드로 관련 경험 탐색

## 프로젝트 범위

프로젝트는 사용자의 커리어 기록을 재사용 가능한 경험 자산으로 바꾸는 것을 목표로 합니다.

범위에 포함되는 내용:

- 커리어 기록 수집
- 경험 정보 구조화
- 경험 근거와 출처 관리
- 검색 가능한 경험 저장소 구성
- API 기반 서비스 구조

범위에서 제외되는 내용:

- 자동 합격 가능성 예측
- 기업별 합격 보장 판단
- 사용자의 동의 없는 외부 문서 수집
- 원문 근거 없는 경험 생성
